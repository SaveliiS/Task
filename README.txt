Для компиляции была использована версия g++ (GCC) 15.1.0
(Версию можно уточнить командой g++ --version)

1. Перейти в директорию, где лежат исполняемые файлы
   программы и тестовые примеры;
2. Скомпилировать командой g++ app.cpp func.cpp -o task;
3. Запустить программу командой task.exe test_file.txt
   Где task.exe  - скомпилированная программа
   test_file.txt - файл с входными данными.
